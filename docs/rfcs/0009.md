## Problem

Фокусный режим удобен для набора текста, но иногда нужно быстро увидеть весь документ целиком
без выхода из редактора.

## Goal

Добавить post-MVP режим просмотра полного файла со скроллингом и безопасным возвратом в фокусный режим.

## Design

### Базовое поведение

- Отдельная горячая клавиша открывает режим preview (конкретный биндинг фиксируется в реализации).
- Режим preview read-only: редактирование отключено.
- Поддерживается вертикальный скроллинг.

### Возврат

- При выходе из preview редактор возвращается в фокусный режим на прежнюю позицию курсора.
- Текущие настройки `lines_before/lines_after` сохраняются.

### UI

- В статус-строке отображается явный индикатор режима preview.
- Позиция в файле остаётся видимой (номер строки/столбца).

## Alternatives

1. **Увеличивать контекст в фокусном режиме до больших значений**
   - Плюсы: не нужен отдельный режим.
   - Минусы: ухудшает основной UX и не заменяет полноценный обзор.

2. **Открывать внешний pager**
   - Плюсы: быстро реализуется.
   - Минусы: потеря целостного UX и сложнее возврат к позиции.

## Voting

Предлагаемое решение: принять отдельный preview-режим как post-MVP расширение.

Статус голосования: `pending`.

## Migration

Миграция не требуется.
